Instant Multi User Todo List API

GET /
- Return this page

GET /me
- Require an Authorization header of Bearer TOKEN
- Return the me object: { id, name, username, token }

GET /lists
List all the todo lists the user has access to
- Require an Authorization header of Bearer TOKEN
- Return a list of todo list: [{ id: "", name: "", userIds: ["", "", ...] }, ...]

POST /action/signup
Signup to create a new account
- Body { name: "", username: "", password: "" } (all properties are required)
  - password must be at least 6 characters
  - username must be at least 3 characters and only contains letters, numbers, - and _
- Return an object with the token of the created user: { token: "" }

POST /action/login
Login using username and password
- Body { username: "", password: "" } (all properties are required)
- Return an object with the token of the user: { token: "" }

POST /action/create-list
Create a new Todo List
- Require an Authorization header of Bearer TOKEN
- Body { name: "" } (name is required and must have at least 1 character)
- Return an object with the id of the created list: { id: "" }

POST /action/add-todo
Add a todo to an existing list
- Require an Authorization header of Bearer TOKEN
- Body { listId: "", name: "", done: true } (done is optional)
  - name must have at least 1 character 
  - listId must be a valid list id 
- Return an object with the id of the created todo: { id: "" }

POST /action/set-todo-done
Set the done property of a todo
- Require an Authorization header of Bearer TOKEN
- Body { listId: "", todoId: "", done: true } (all properties are required)
- Return an object with the id of the created todo: { id: "" }

POST /action/set-todo-done
Invite a user to give him acces to a todo list
- Require an Authorization header of Bearer TOKEN
- Body { listId: "", username: "" } (all properties are required)
- Return a 204 (No Content) response
